Multi-Converter

Convert fractions, decimals, or time (1 hour = 60 minutes)

Note: If conversion doesn't work, try running a local server: python -m http.server 8000 then open http://localhost:8000/index.html

Result will appear here

Examples:

  • Fractions: 1/2, 3/4, 5 / 8
  • Decimals: 0.5, 0.25, 0.333
  • Repeating: 0.333 (detects as 1/3)
